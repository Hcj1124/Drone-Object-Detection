# 無人機空拍影像移動物件辨識

這是一項由國立中興大學資管系開發的畢業專題研究，旨在解決無人機在複雜軍事環境下，針對微小目標與低對比影像進行移動物件偵測的技術難題。

### Project Overview
本研究提出一套基於 RGB 影像的移動物件識別流程。透過兩階段效能優化策略，我們成功提升了在複雜背景下偵測動態目標的穩定性與準確度。

### Tech Stack
* **語言**: Python
* **核心框架**: PyTorch, YOLOv7
* **影像處理**: OpenCV, NumPy
* **開發工具**: Jupyter Notebook, VS Code

### Key Findings
* **效能提升**: 透過超參數調整（Hyperparameter Tuning），將召回率 (Recall) 從 47.5% 提升至 65.7%。
* **背景分類處理**: 引入背景感知分類策略，將精確率 (Precision) 從 62.9% 提升至 83.8%，F1-score 提升了約 19.5%。
* **數據規模**: 處理 3521 張無人機空拍影像，並建立自動化處理 Pipeline。
* **特徵融合**: 整合光流向量場 (Optical Flow) 技術，解決了在無人機空拍低對比度與複雜背景下，目標物識別率低落的問題。

### Results
本研究比較了優化前後的模型性能，詳細數據如下表所示：

| Metric | Pre-Optimization | Post-Optimization | Improvement |
| :--- | :---: | :---: | :---: |
| **Precision** | 62.9% | 83.8% | +20.9% |
| **Recall** | 47.5% | 65.7% | +18.2% |
| **F1-Score** | 54.1% | 73.6% | +19.5% |
